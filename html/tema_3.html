Målgruppe

Indenfor UX fokuserede vi på research, målgruppe analyse samt måling og test af indholdet. Vi brugte metoden 'user research', hvor man først designer, herefter tester, og sidst implementerer på sit site. Vi brugte metoder som user stories, lavede værdiord og moodboards samt styletiles og opsætning af vores brugte farver som 'styles' i Figmas bibliotek.

Vi lavede en masse øvelser og lærte b.la. Om de forskellige plugins i Figma med bla.
